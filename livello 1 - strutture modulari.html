Livello 1 – Strutture Modulari

In questo livello approfondiamo l'organizzazione del codice Python tramite funzioni, gestione degli errori e importazione di moduli. Imparerai a scrivere codice più ordinato, riutilizzabile e robusto.

1. Funzioni

Le funzioni servono a racchiudere blocchi di codice che eseguono operazioni specifiche. Si definiscono con def e si richiamano con il loro nome.

def greet(name):
    print(f"Hello, {name}!")

greet("Alice")

Parametri e ritorni

def add(a, b):
    return a + b

result = add(3, 5)
print(result)

Argomenti di default

def greet(name="Guest"):
    print(f"Welcome, {name}!")

greet()
greet("Bob")

*args e **kwargs

def print_args(*args):
    for arg in args:
        print(arg)

print_args("a", "b", "c")

def print_kwargs(**kwargs):
    for key, value in kwargs.items():
        print(key, value)

print_kwargs(name="Alice", age=25)

Esercizi

  • Scrivi una funzione che calcola l'area di un rettangolo (base * altezza).
  • Scrivi una funzione che accetta un numero qualsiasi di argomenti e ne stampa la somma.

2. Scope delle variabili

Una variabile dichiarata dentro una funzione ha scope locale. Le variabili dichiarate fuori da tutte le funzioni hanno scope globale.

x = 10

def example():
    x = 5  # locale
    print(x)

example()
print(x)  # globale

Esercizio

  • Prova a modificare una variabile globale da dentro una funzione usando la parola chiave global.

3. Gestione degli errori

Python permette di gestire eccezioni usando try / except. È utile per evitare che il programma si blocchi in caso di errore.

try:
    x = int(input("Enter a number: "))
    print(10 / x)
except ValueError:
    print("Invalid input: not a number")
except ZeroDivisionError:
    print("You can't divide by zero")

Uso di finally

try:
    file = open("data.txt")
    # ...
except FileNotFoundError:
    print("File not found")
finally:
    print("Execution finished")

Esercizi

  • Scrivi un programma che chiede un numero all'utente e stampa il suo inverso, gestendo eccezioni.
  • Simula un errore con una variabile non definita e intercettalo con except NameError.

4. Importazione di moduli

Un modulo è un file Python che contiene definizioni, funzioni o variabili. Puoi importare moduli interni o librerie esterne.

import math
print(math.sqrt(25))
from random import randint
print(randint(1, 10))

Esercizi

  • Importa il modulo datetime e stampa la data odierna.
  • Usa from ... import ... per importare solo choice dal modulo random e usalo su una lista di nomi.

Verifica di fine livello

  • So scrivere e richiamare funzioni, anche con parametri variabili.
  • So distinguere tra scope locale e globale.
  • So gestire errori ed eccezioni comuni con try / except.
  • So importare moduli della libreria standard e usare le loro funzioni.
