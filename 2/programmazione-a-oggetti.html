Livello 2 – Programmazione a Oggetti

In questo livello impari a usare le classi e gli oggetti in Python. La programmazione orientata agli oggetti (OOP) ti permette di strutturare il codice in modo più modulare, riutilizzabile e adatto a progetti complessi.

1. Classi e oggetti

Una classe è un modello (template). Un oggetto è un'istanza concreta di quella classe.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print(f"Hi, I'm {self.name} and I'm {self.age} years old.")

person1 = Person("Alice", 30)
person1.greet()

Esercizi

  • Crea una classe Car con attributi brand e year. Aggiungi un metodo che stampa una descrizione.

2. Incapsulamento

In Python, gli attributi privati si definiscono con un underscore. Non sono davvero "privati", ma è una convenzione per indicare che non vanno modificati direttamente.

class BankAccount:
    def __init__(self, balance):
        self._balance = balance

    def deposit(self, amount):
        self._balance += amount

    def get_balance(self):
        return self._balance

3. Ereditarietà

Una classe può ereditare da un'altra per riutilizzarne attributi e metodi.

class Animal:
    def speak(self):
        print("Some sound")

class Dog(Animal):
    def speak(self):
        print("Woof")

d = Dog()
d.speak()

Esercizio

  • Crea una classe Employee e una classe Manager che eredita da Employee.

4. Polimorfismo

Con il polimorfismo, puoi scrivere codice che funziona con oggetti di classi diverse ma con metodi comuni.

class Cat:
    def speak(self):
        print("Meow")

animals = [Dog(), Cat()]
for animal in animals:
    animal.speak()

5. Metodi speciali

I metodi speciali (o d'uso interno) iniziano e finiscono con doppio underscore.

class Book:
    def __init__(self, title):
        self.title = title

    def __str__(self):
        return f"Book: {self.title}"

b = Book("1984")
print(b)

6. Decoratori di classe

Python supporta @staticmethod, @classmethod e @property per definire comportamenti particolari.

class Math:
    @staticmethod
    def add(a, b):
        return a + b

print(Math.add(3, 4))

Esercizi

  • Crea una classe Circle con un attributo radius e un metodo @property per calcolare l'area.
