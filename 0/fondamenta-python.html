Livello 0 – Le Fondamenta di Python

In questo livello impari le basi assolute del linguaggio Python: la sintassi, i tipi di dato fondamentali, il flusso del programma e le principali strutture dati.

1. Variabili e tipi primitivi

Una variabile è un nome che fa riferimento a un valore in memoria. In Python non serve specificare il tipo della variabile: lo determina automaticamente al momento dell'assegnazione.

name = "John"
age = 30
height = 1.75
has_license = True

Tipi principali:

  • int: numeri interi
  • float: numeri decimali
  • str: stringhe di testo
  • bool: valori logici
print(type(name))   # <class 'str'>
print(type(age))    # <class 'int'>

Esercizi

  • Crea una variabile name con il tuo nome.
  • Crea variabili age, height e has_license.
  • Stampa ogni variabile e il relativo tipo con type().

2. Operatori

a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a // b)
print(a % b)
print(a ** b)
print(a == b)
print(a != b)
print(a > b)
print(a <= b)
x = True
y = False

print(x and y)
print(x or y)
print(not x)

Esercizi

  • Dichiara due numeri a e b e prova le operazioni aritmetiche.
  • Confronta a e b con gli operatori logici.
  • Prova gli operatori and, or e not.

3. Stringhe

text = "  Python is powerful  "

print(text.strip())
print(text.lower())
print(text.upper())
print(text.replace("powerful", "easy"))
name = "Alice"
age = 25

print("My name is " + name + " and I am " + str(age) + " years old.")
print(f"My name is {name} and I am {age} years old.")

Esercizi

  • Crea una stringa name e stampala in maiuscolo.
  • Usa strip() per rimuovere gli spazi.
  • Costruisci una frase con un f-string.

4. Input e output

name = input("What is your name? ")
print(f"Nice to meet you, {name}!")
age = int(input("How old are you? "))

Esercizi

  • Chiedi all’utente il nome e saluta.
  • Chiedi l’età e calcola quanti anni avrà tra 10 anni.

5. Controllo del flusso

age = 18

if age < 18:
    print("You are underage.")
elif age == 18:
    print("You just turned 18!")
else:
    print("You are an adult.")
for i in range(5):
    print(i)
counter = 0
while counter < 3:
    print("Counter:", counter)
    counter += 1

Esercizi

  • Stampa i numeri da 1 a 10 con for.
  • Verifica se un numero è pari o dispari.
  • Fai indovinare all’utente una parola segreta con while.

6. Strutture dati fondamentali

fruits = ["apple", "banana", "kiwi"]
fruits.append("pear")
print(fruits[1])
coordinates = (10, 20)
print(coordinates[0])
numbers = {1, 2, 3, 2}
print(numbers)
person = {"name": "Alice", "age": 25}
print(person["name"])
person["age"] = 26

Esercizi

  • Crea una lista con 3 frutti e stampali in un ciclo.
  • Crea una tupla con tre numeri.
  • Crea un dizionario con nome ed età, poi aggiorna l’età.

Verifica finale

  • So dichiarare variabili e tipi di dato.
  • So usare operatori aritmetici, logici e di confronto.
  • So lavorare con stringhe, input e output.
  • So usare if, for, while.
  • So gestire liste, tuple, set e dizionari.

Progetto pratico

Scrivi un programma che:

  1. Chiede all'utente il nome, l'età e il colore preferito.
  2. Salva queste informazioni in un dizionario.
  3. Stampa una frase come:
    Hello Alice! You are 25 years old and your favorite color is blue.
